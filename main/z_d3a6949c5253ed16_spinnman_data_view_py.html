175 """ 

176 Read some areas of memory (usually SDRAM) from the board. 

177 

178 Syntactic sugar for `get_transceiver().read_memory()`.

180 :param x: 

181 The x-coordinate of the chip where the memory is to be read from 

182 :param y: 

183 The y-coordinate of the chip where the memory is to be read from 

184 :param base_address: 

185 The address in SDRAM where the region of memory to be read starts 

186 :param length: The length of the data to be read in bytes 

187 :param cpu:

209 @classmethod 

210 def write_memory( 

211 cls, x: int, y: int, base_address: int, data: Union[

213 n_bytes: Optional[int] = None, offset: int = 0, 

214 cpu: int = 0) -> Tuple[int, int]: 

215 """ 

216 Write to the SDRAM on the board. 

217 

218 Syntactic sugar for `get_transceiver().write_memory()`. 

219 

220 :param x: 

221 The x-coordinate of the chip where the memory is to be written to 

222 :param y: 

223 The y-coordinate of the chip where the memory is to be written to 

224 :param base_address: 

225 The address in SDRAM where the region of memory is to be written 

226 :param data: The data to write. Should be one of the following: 

227 

228 * An instance of RawIOBase 

229 * A bytearray/bytes 

230 * A single integer - will be written in little-endian byte order 

231 * A filename of a data file 

232 :param n_bytes: 

233 The amount of data to be written in bytes. If not specified: 

234 

235 * If `data` is an RawIOBase, an error is raised 

236 * If `data` is a byte string (bytearray or bytes), the length of\ 

237 the byte string will be used 

238 * If `data` is an int, 4 will be used 

239 * If `data` is a str, the length of the file will be used 

240 :param offset: The offset from which the valid data begins 

241 :param cpu: The optional CPU to write to 

242 :return: The number of bytes written, the checksum (0 if get_sum=False) 

243 :raises ~spinn_utilities.exceptions.SpiNNUtilsException: 

244 If the transceiver is currently unavailable 

245 :raise SpinnmanIOException: 

246 * If there is an error communicating with the board 

247 * If there is an error reading the data 

248 :raise SpinnmanInvalidPacketException: 

249 If a packet is received that is not in the valid format 

250 :raise SpinnmanInvalidParameterException: 

251 * If `x, y` does not lead to a valid chip 

252 * If a packet is received that has invalid parameters 

253 * If `base_address` is not a positive integer 

254 * If `data` is an RawIOBase but `n_bytes` is not specified 

255 * If `data` is an int and `n_bytes` is more than 4 

256 * If `n_bytes` is less than 0 

257 :raise SpinnmanUnexpectedResponseCodeException: 

258 If a response indicates an error during the exchange 

259 """
