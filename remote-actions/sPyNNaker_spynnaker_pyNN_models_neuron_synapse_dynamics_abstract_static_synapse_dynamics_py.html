38 def get_static_synaptic_data( 

39 self, connections, connection_row_indices, n_rows, 

40 post_vertex_slice, n_synapse_types, max_n_synapses): 

41 """ Get the fixed-fixed data for each row, and lengths for the\ 

42 fixed-fixed parts of each row. 

43 

44 Data is returned as an array made up of an array of 32-bit words for\ 

45 each row for the fixed-fixed region. The row into which connection\ 

46 should go is given by `connection_row_indices`, and the total number\ 

47 of rows is given by `n_rows`. 

48 

49 Lengths are returned as an array made up of an integer for each row,\ 

50 for the fixed-fixed region. 

51 

52 :param ~numpy.ndarray connections: The connections to get data for 

53 :param ~numpy.ndarray connection_row_indices: 

54 The row into which each connection should go 

55 :param int n_rows: The number of rows to write

57 The slice of the post vertex to generate for 

58 :param int n_synapse_types: The number of synapse types 

59 :param int max_n_synapses: The maximum number of synapses to generate 

60 :return: (ff_data, ff_size) 

61 :rtype: tuple(list(~numpy.ndarray), ~numpy.ndarray) 

62 """
