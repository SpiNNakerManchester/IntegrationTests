35class SQLiteDB(object): 

36 """ 

37 General support class for SQLite databases. This handles a lot of the 

38 low-level detail of setting up a connection. 

39 

40 Basic usage (with the default row type):: 

41 

42 with SQLiteDB("db_file.sqlite3") as database: 

43 with database.transaction() as cursor: 

44 for row in cursor.execute("SELECT thing FROM ..."): 

45 print(row["thing"]) 

46 

47 This class is designed to be either used as above or by subclassing. 

48 See the `SQLite SQL documentation <https://www.sqlite.org/lang.html>`_ for 

49 details of how to write queries, and the Python :py:mod:`sqlite3` module 

50 for how to do parameter binding. 

51 

52 .. note:: 

53 If you plan to use the WAL journal mode for the DB, you are 

54 *recommended* to set this up in the DDL file via:: 

55 

56 PRAGMA journal_mode=WAL; 

57 

58 This is because the journal mode is persistent for database. 

59 For details, see the 

60 `SQLite documentation <https://www.sqlite.org/wal.html>`_ on 

61 the write-ahead log. 

62 """
